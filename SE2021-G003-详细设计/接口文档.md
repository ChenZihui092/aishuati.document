# 管理员端

## 1. 登录

- 接口请求地址： /admin/login

  ```
  {
  	"userAccount": "admin", //用户名
  	"password": "" ,
  }
  ```

  

- 返回参数

  ```
  {
  	"code": 200,
  	"message": "",
  	
  }
  ```

  

## 2. 注册

- 接口请求地址：/admin/reg

  ```json
  {
      "userAccount":"",
      "password":"",
      "rep_passord":"",
      "name":"",
      "email":"",
      "sex":"",
      "telephone":"",
      "state":"", //权限
  }
  ```

  

- 返回参数

  ```json
  {
      "code":"200",
      "message":"",
  }
  ```
