# 管理员端

## 1. 登录

- 接口请求地址： /admin/login

  ```
  {
  	"userAccount": "admin", //用户名
  	"password": "" ,
  }
  ```

  

- 返回参数

  ```
  {
  	"code": 200,
  	"message": "",
  	
  }
  ```
