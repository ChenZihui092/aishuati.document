

# 管理员端

## 1. 登录

- 接口请求地址： /admin/login

  ```
  {
  	"userAccount": "admin", //用户名
  	"password": "" ,
  }
  ```

  

- 返回参数

  ```
  {
      "code": 200,
      "msg": "退出成功"
  }
  ```

## 10 . 题目编创

  ```Json
  {

      "problemType":"",
      "difficult":"",
      "info_text_content":"",  //题面
      "correct":"",      //正确答案
      "choice_A":"",
      "choice_B":"",
      "choice_C":"",
      "choice_D":"",
  }
  ```

  主观题 type = 2

  ```Json
  {
   	"subjectName":"",
      "problemType":"",
      "difficult":"",
      "info_text_content":"",
      "reference":""
  }
  ```

- 返回参数

  ```Json
  {
      "code":200,
      "msg":"添加成功"
  }
  ```
